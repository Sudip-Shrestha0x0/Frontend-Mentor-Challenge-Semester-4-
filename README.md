## Overview

This project is a solution to the News homepage challenge from Frontend Mentor. The challenge focuses on building a responsive news website homepage using CSS Grid layout techniques.

### The Challenge

Users are able to:
- View the optimal layout for the interface depending on their device's screen size
- See hover and focus states for all interactive elements on the page
- Navigate through a mobile menu on smaller screens

### Key Features

- **Responsive Design:** Optimized layouts for mobile (375px), tablet, and desktop (1440px) screens
- **CSS Grid Layout:** Main content area uses a three-column grid system that adapts to different screen sizes
- **Interactive Elements:** Hover states for navigation links, buttons, and article titles
- **Mobile Menu:** Slide-in navigation menu with overlay for mobile devices
- **Accessibility:** Semantic HTML structure and ARIA labels for better screen reader support

### Layout Structure

The layout uses CSS Grid to create a responsive design:
- Desktop: 3-column grid with hero section, new articles sidebar, and featured articles
- Mobile: Single-column stacked layout for optimal mobile viewing
